# Limits

A terminal app that displays OS configuration and limits helpful to debug issues.

## Usage 

- Directly run as a script:

```shell
uv run limits.py
```

- Install dependencies and run 

```shell
uv sync
```

```shell
uv run limits.py
```

# Key bindings

Press `q` to quit
Press `r` to refresh 

# Dependencies

Optional

- textual-dev is added to debug
- ruff for formatting

This allows to run in debug mode

```shell
textual run limits.py
```
